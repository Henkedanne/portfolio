# Henrik Danielssons portfolio

This is my portfolio. I'm using the boilerplate that I built. 

## Instructions
### Installation
Install node.js:
```sh
$ npm install
```
Install Gulp.js:
```sh
$ npm install gulp
```
---
### Run
Start Browser sync and Sass compile:
```sh
$ gulp watch
```
---
 
Learned from "https://css-tricks.com/gulp-for-beginners/".
